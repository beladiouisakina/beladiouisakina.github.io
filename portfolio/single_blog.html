London Underground System Route Planner

Result:
Successfully identified the Minimum Spanning Tree (MST) of the London Underground network using Kruskal's algorithm, minimizing total connection costs between stations and ensuring optimal routing efficiency.
Provided insights into potential route optimizations to enhance the system's overall performance.
Strategy:
Leveraged Kruskal’s algorithm to find the MST of the network, focusing on minimizing the cost of connections between stations.
Applied union-find (disjoint-set) techniques to efficiently detect and connect the minimum-cost edges.
Analyzed the graph structure of the London Underground system to ensure all stations were connected in the most cost-effective manner.
Approach:
Represented the London Underground system as a weighted graph where edges represented connections between stations and weights reflected travel costs or distances.
Implemented Kruskal’s algorithm to iteratively add edges with the least weight while avoiding cycles.
Conducted time complexity analysis and optimized the algorithm for large-scale network graphs.
Compared the MST with existing routes to identify areas for improvement and potential enhancements.
